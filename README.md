# Quotes (Web App)

Educational, simple CRUD application with Express and MongoDB.

#### Using:  
- JavaScript ES6, NodeJS, Express, MongoDB, Mongo Atlas, CRUD 
- EJS (Embedded JavaScript), Nodemon, Body-Parser
